UI-Angularjs-Framework

Este framework se creó con la finalidad de automatizar tareas de desarrollo y también organizar archivos y optimizar tareas para despliegue en producción.

Estructura

  • assets
  • dev
  • prod
  • config
  • node_modules

ASSETS

Dentro de ASSETS se encuentra todo el código fuente.

  • app

    Todos los HMLT y Js relacionados con secciones o componentes.

  • icons

    Toda la iconografía relacionada con icomoon

  • js

    Cualquier librería externa en js.

  • styles

    todos los estilos escritos en SASS que no estén relacionados con componentes.

DEV

Dentro de DEV se encuentran los archivos ya unificados que levanta el server local

  • css

    Todos los estilos en .css

  • fonts

    Iconos de icomoon

  • js

    Todos los js.

  • templates

    Los partials html

  • index.html

    Es el index de todo servidor, lo primero que levanta.

CONFIG

Los archivos de configuración para Gulp.

NODE_MODULES

Dependencias de node

ARCHIVOS

  • .gitignore

    Definiciones de las rutas que no se desea trackear en GIT

  • Gulpfile.js

    Todas las tareas de GULP.

  • package.json

    Todas las dependencias de node con sus versiones.
